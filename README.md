# cam - a unix command for viewing images
## Installation

    git clone https://github.com/itchyny/cam
    cd ./cam
    autoreconf -i
    ./configure
    make
    sudo make install

## Example

    cam test.jpg
    cam *
    cam -h 50 test.jpg
    cam -C *
    cam -R *

## Manual

    man cam

## Name
The command name `cam` has its root in the word `camera`.
